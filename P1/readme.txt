En aquesta pràctica, utilitzant les classes Myocket i MyServerSocket, que son molt similars a Socket i ServerSocket, amb la diferència que les nostres encapsulen les excpecions, hem fet un xat per terminal. La explicació de cada classe es detalla al inci de la mateixa. 
Per executar, el model és el següent:
Creem un servidor: java Server <port>
Ens conectem al servidor amb un client: java Client <host> <port>
